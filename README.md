# url_short

A nice url shortener

---

## How to deploy

    git clone https://github.com/Manthee1/url_short.git
    cd url_short
    npm install

* Starting on Windows

        npm start
        or
        node server.js

* Starting on Linux

        sudo npm start
        or
        sudo node server.js

Make sure you have both nodejs and mongodb installed!  
If you don't know how to install them. Google is your friend.
